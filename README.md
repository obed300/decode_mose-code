## 💻 Getting Started <a name="getting-started"></a>

## To get a local copy up and running, follow these steps.

### Setup

To get this repository to your desired folder:

```sh
  cd my-folder
  gitclone https://github.com/ambrose-kibet/decode_morse_code.git
```

### Running the app

To get app running:

```sh
    bundle install
    ruby main.rb
```
